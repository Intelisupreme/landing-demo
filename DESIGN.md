# Дизайн-система: Intelisupre

## 1. Философия бренда (Brand Identity)
**Intelisupre** объединяет передовые технологии (AI, SaaS) с простотой и эффективностью для бизнеса. Дизайн лендинга должен быть чистым, интуитивно понятным и внушающим доверие. Плавные линии интерфейса должны резонировать с геометрией логотипа, исключая агрессивные острые углы.

---

## 2. Цветовая палитра (Color Palette)

Основной акцент сделан на цвета, извлеченные из вашего логотипа. Для SaaS-платформ крайне важно иметь контрастное разделение между фоном, поверхностями карточек и текстом для обеспечения высокой читаемости.

### Брендовые цвета (Brand Colors)
| Название | HEX | RGB | Применение |
| :--- | :--- | :--- | :--- |
| **Brand Blue** | `#299EBD` | `41, 158, 189` | Иконки, текстовые ссылки, акцентные элементы |
| **Brand Green** | `#38A98E` | `56, 169, 142` | Индикаторы успеха, чекмарки, графики |
| **Brand Gradient** | - | - | `linear-gradient(135deg, #299EBD 0%, #38A98E 100%)` (Кнопки CTA) |

### Светлая тема (Light Mode)
Обеспечивает легкий, корпоративный и свежий вид.

| Элемент | HEX | Описание |
| :--- | :--- | :--- |
| **Background** | `#F8FAFC` | Основной фон страницы (очень светлый серо-голубой) |
| **Surface** | `#FFFFFF` | Фон карточек тарифов, модальных окон, меню |
| **Text Primary** | `#0F172A` | Основной текст и заголовки |
| **Text Secondary** | `#64748B` | Подзаголовки, плейсхолдеры, описания функций |
| **Border / Line**| `#E2E8F0` | Разделители, деликатные рамки карточек |

### Темная тема (Dark Mode)
Для AI-продуктов темная тема создает премиальный и технологичный образ. Базируется на глубоких сине-серых оттенках, а не на чистом черном.

| Элемент | HEX | Описание |
| :--- | :--- | :--- |
| **Background** | `#0B1120` | Основной фон страницы |
| **Surface** | `#1E293B` | Фон карточек и элементов поверх основного фона |
| **Text Primary** | `#F8FAFC` | Основной текст и заголовки (высокий контраст) |
| **Text Secondary** | `#94A3B8` | Второстепенный текст |
| **Border / Line**| `#334155` | Разделители, границы модальных окон |

## 3. Типографика (Typography)

Шрифтовая пара подобрана с учетом технологичности (AI) и необходимости читать большие объемы текста (SaaS).

* **Шрифт заголовков (Headings):** `Outfit` — современный, геометрический гротеск. Отлично подчеркивает "технологичность".
* **Основной шрифт (Body):** `Inter` — неоспоримый стандарт для веб-интерфейсов, гарантирующий идеальную читаемость.

## 4. Сетка (Grid System)

Лендинг использует гибкую 12-колоночную сетку для выравнивания контента по центру экрана.

| Брейкпоинт | Ширина экрана | Контейнер (Max-width) | Колонки | Gutter (Межколоночный) | Margin (Внешние поля) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Mobile** | < 768px | 100% | 4 | 16px | 20px |
| **Tablet** | 768px - 1023px| 720px | 8 | 24px | 32px |
| **Desktop**| 1024px - 1439px| 960px | 12 | 24px | Auto |
| **Wide** | > 1440px | 1280px | 12 | 32px | Auto |

---

## 5. Пространственная система (Spacing)

Отступы строятся на классической **8-пиксельной сетке** (8pt grid). Это гарантирует визуальный ритм и гармонию элементов.

* **4px / 8px:** Внутренние микро-отступы (между иконкой и текстом, внутри инпутов).
* **16px / 24px:** Внутренние отступы (padding) в карточках фич, алертах.
* **32px / 48px:** Расстояния между элементами одной секции (например, между заголовком секции и сеткой карточек).
* **64px / 96px / 128px:** Глобальные вертикальные отступы между крупными логическими блоками Landing Page (Hero -> Features -> Pricing).

## 6. Радиусы скругления (Border Radius)

Поскольку в основе логотипа Intelisupre лежат мягкие изгибы и отсутствие острых углов, интерфейс должен отражать эту пластику. Полностью прямые углы (`0px`) не используются.

| Переменная | Значение | Применение |
| :--- | :--- | :--- |
| `--radius-sm` | `6px` | Чекбоксы, маленькие теги, тултипы. |
| `--radius-md` | `12px` | Поля ввода (инпуты), выпадающие списки, стандартные кнопки. |
| `--radius-lg` | `24px` | Большие карточки SaaS-тарифов, плашки с отзывами, модальные окна. |
| `--radius-full` | `9999px` | Главные кнопки призыва к действию (CTA), аватарки пользователей. |

## 7. Компоненты и Тени (Elevation & UI Elements)

### Кнопки
* **Primary (Основная):** Заливка — `Brand Gradient`, текст — белый, скругление — `--radius-full`. При наведении — легкое увеличение яркости и добавление цветной тени.
* **Secondary (Вторичная):** Фон — прозрачный (или `Surface`), граница — `2px solid #299EBD`, текст — `#299EBD`. 
* **Ghost (Третичная):** Текст — `Text Secondary`, без фона. При наведении появляется полупрозрачный серый/голубой фон.

### Тени (Shadows)
В светлой теме тени имеют легкий холодный подтон, чтобы не казаться "грязными". В темной теме вместо теней для разделения слоев используется изменение цвета фона (`Surface`) и рамки (`Border`).

* **Shadow SM:** `0 2px 4px rgba(15, 23, 42, 0.05)` — Инпуты, небольшие кнопки.
* **Shadow MD:** `0 4px 6px -1px rgba(15, 23, 42, 0.08)` — Навигационная панель (header) при скролле, выпадающие списки.
* **Shadow LG (Accent):** `0 10px 20px -3px rgba(41, 158, 189, 0.15)` — Выделение главного тарифа в прайсинге, акцентные плавающие элементы (используется брендовый синий цвет для генерации тени).
